Пожалуйста, создайте структуру базы данных с индексами для компании, занимающейся подбором персонала. БД должна иметь возможность хранить персональные данные соискателей, образование, уровень подготовки в различных областях знаний, опыт работы и так далее. Данные по уровню подготовки могут иметь многоуровневые связи. Например:
1) Language skills
a.	English 
	i.	Writing
	ii.	Reading
	iii.	…
b.	German
c.	French
2) Computer skills
	a.Office software
		i.	…
	b.Computer languages
		i.	Compilers
			1.	C
			2.	C++
			3.	Java
			4.	Assembler
		ii.	Scripting languages
			1.	VB script
			2.	TCL
			3.	Java script
			4.	…
			5.	Perl
	c.Hardware
	d.Databases
	e.Operation Systems
3) …
4) Бухгалтерский учет
 
Эта задача предполагает более высокий уровень свободы для ее решения. Представьте себя на месте сотрудника агентства и попробуйте сделать БД удобной для работы при подборе кадров от уровня профессора или директора завода до охранника детского сада.
